gh_ordinalize

Ordinalize.

- (NSString *)gh_ordinalize

Return Value

Ordinal string for integer.

Discussion

 0 => nil
 1 => "1st"
 2 => "2nd"
 3 => "3rd"
 4-9 -> "4th", "5th", ...
 Ending in 11, 12 or 13 => "111th", "212th", ...
